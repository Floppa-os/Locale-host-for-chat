Introduction

Welcome to the Locale-host-for-chat Wiki — your go‑to resource for setting up and using the local chat host.

This wiki provides documentation, setup guides, and answers to common questions.

Setup Guide

  1. Clone the repository: git clone https://github.com/your-username/locale-host-for-chat.git
  2. Navigate to the project directory: cd locale-host-for-chat
  3. Install dependencies: npm install
  4. Start the server: npm start
  5. Open http://localhost:3000 in your browser.

FAQ

How do I change the port?
Edit the .env file and set PORT=your-port.
Can I use this without Node.js?
No, Node.js is required for the server to run.
Where are the chat logs stored?
Logs are saved in the ./logs/ directory.
